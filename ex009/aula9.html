A tag <ol> (OL - Ordened List) possui o parâmetro type para configurar o tipo de marcador e o parâmetro start para determinar o início da contagem

Possíveis valores para o parâmetro type:

  1. '1' - Valor padrão, lista numerada (1, 2, 3, 4, 5...)
  2. 'A' - Listas alfabéticas em maiúsculas (A, B, C, D, E...)
  3. 'a' - Listas alfabéticas em minúsculas (a, b, c, d, e...)
  4. 'I' - Listas com algarismos romanos em maiúsculas (I, II, III, IV, V...)
  5. 'i' - Listas com algarismos romanos em minúsculas (i, ii, iii, iv, v...)

A tag <ul> (UL - Unordered List) tambêm possui o parâmetro type

Possíveis valores para o parâmetro type (ul):

  • disc - <valor default> Uma bola preta preenchida com a cor preta
  • circle - Um círculo com a borda preta sem preenchimento.
  • square - Um pequeno quadrado totalmente preenchido com a cor preta
