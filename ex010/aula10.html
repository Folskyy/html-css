Parâmetros<a>:

  • href: Referência Hipertexto
  • hreflang: Idioma principal do site referenciado
  • target: Controla onde o site vai abrir
    • _self: (default) Abre o link na janela atual
    • _blank: Vai abrir o link em uma nova janela em branco
    • _top: Vai desfazer todos os frames e abrir o destino no navegador completo
    • _parent: Abre o link no quadro pai do quadro onde o link foi clicado se houver um
  • rel: Indica qual é a natureza do destino
    • next: Indica que o link é para a próxima parte do documento atual
    • prev: Indica que o link é para a parte anterior do documento atual
    • author: Indica que é um link para o site do autor do artigo atual
    • external: Indica que é um link para outro site que não faz parte do site
    • nofollow: Indica que é um link para um site não endossado, como um link pago
  • download: O valor é o nome do arquivo a ser baixado. Mas pra isso é preciso que o atributo type esteja configurado para indicar o tipo do arquivo:
    • application/zip
    • text/html
    • text/css
    • text/javascript
    • video/mp4
    • video/H264
    • video/JPEG
    • audio/aac
    • audio/mpeg
    • font/ttf
    • image/jpeg
    • image/png
